# 🏥 Hospital Emergency Room Dashboard

## 📌 Project Overview
The **Hospital Emergency Room Dashboard** is an interactive healthcare analytics project designed to monitor emergency room performance and patient flow.

This dashboard helps analyze:
- Total number of patients
- Average waiting time
- Patient satisfaction score
- Admission vs non-admission status
- Age group analysis
- Gender-wise patient distribution
- Department referral trends
- Monthly and yearly patient insights
-The goal of this project is to transform raw hospital data into meaningful visual insights for operational and performance analysis.

## 🎯 Objective
The objective of this dashboard is to help hospital staff and management:
- Monitor patient traffic
- Improve emergency room efficiency
- Reduce waiting time
- Understand patient demographics
- Analyze department workload
